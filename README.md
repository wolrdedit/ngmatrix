## Development

### Local Development

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/ngmatrix.git
   cd ngmatrix
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Build the library:

   ```bash
   ng build ngmatrix
   ```

4. Start the demo application:
   ```bash
   ng serve demo
   ```
